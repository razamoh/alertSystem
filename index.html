PosAlert Demo

A customizable alert system for POS applications

Basic Alerts

$.posAlert().alert({ title: 'Information', message: 'This is a basic alert message.', theme: 'primary' });

Confirmation Dialogs

$.posAlert().confirm({ title: 'Confirm Action', message: 'Are you sure you want to proceed?', confirmText: 'Yes', cancelText: 'No', theme: 'primary', onConfirm: function() { console.log('Confirmed'); }, onCancel: function() { console.log('Cancelled'); } });

Prompt Dialogs

$.posAlert().prompt({ title: 'Enter Information', message: 'Please provide the requested information:', placeholder: 'Type here...', theme: 'primary', onConfirm: function(value) { console.log('Value entered:', value); } });

Toast Notifications

$.posAlert().toast({ message: 'This is a toast notification', type: 'success', // success, error, warning, info duration: 3000 });
